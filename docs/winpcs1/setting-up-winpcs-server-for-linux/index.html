Setting up a Linux(Debian) Server for WinPCS1

This document is a set of steps and instructions I have used to install Debian 12 for the WinPCS1 Couchdb.

Create Bootable USB Drive​

To create a bootable USB for Debian 12 from a Windows 11 computer, you can use Etcher.

  1. First, download the Debian 12 ISO file from the official Debian website
  2. Insert a USB drive with at least 4 GB of storage into your Windows 11 computer.
  3. Open Etcher, select your USB drive, and choose the Debian 12 ISO file.
  4. Click Flash to begin the process of creating the bootable USB.

Install Debian on Server​

For a video guide of installation steps taken, see How to install Debian 12 Bookworm For Linux Beginners (UEFI) Ensure the UEFI is enabled and if Legacy Boot option is available to disable it. Also, disable Secure Boot and Fast Boot.

Ensure that your system's BIOS is configured to prioritize USB as the primary bootable device (F2 Key or F10 for Boot Menu). Then, boot your system using the freshly created Debian 12 bootable USB Drive. Follow the steps below to proceed with the installation of Debian 12, codenamed Bookworm.

STEP 1 : Choose Install Type

From the Debian installer menu, choose the option labeled "Graphical Install" to initiate the installation process of Debian 12.

STEP 2: Language Selection

In this step, you will be prompted to select the language that will be used for both the installation process and the system language in Debian 12. Choose the English language from the available options to proceed with the installation.

STEP 3: Select Your Location

In this step, you will be required to select your location, which will automatically set the appropriate time zone for your Debian 12 installation. Choose United States and click "continue".

STEP 4: Keyboard Configuration

Select American English keyboard layout from the available options and click "continue". Now, some installer components will be loaded from the disk before going to the next step.

STEP 5: Network Configuration

  1. Ensure the server is connected to the network. This will automatically configure IP address by obtaining it from an existing DHCP server in your network (this will be configured later for static IP).

During the installation process, the Debian installer will attempt to automatically configure the IP address by obtaining it from an existing DHCP server in your network. However, if no DHCP server is available, the automatic network autoconfiguration will fail. In such a scenario, you will need to manually configure the IP address settings. Click "continue" to manually configure your network. In the next screen, choose "Configure network manually" option and click "continue".

  • Enter a IP address to your Debian system and click "continue".
  • Enter Netmask and click "continue".
  • Enter Gateway IP and click "continue".
  • Enter Name server (DNS) address and click "continue".
  1. Enter a hostname for your Debian system and click "continue".
  2. Next, enter a Domain name for your Debian system. The domain name helps identify and categorize systems within a network. If you have a specific domain name associated with your network, enter it here. Otherwise, you can leave this field blank. Providing a domain name is optional, and it can be configured or modified later if needed. I am leaving it blank here.

STEP 6: Set Root Password

In this step, leave the root password blank and click "continue". This gives the initial user account the power to become root using the "sudo" command.

Debian Warning: The root user should not have an empty password. If you leave this emplty, the root account will be discabled and the system's initial user account will be given the power to become root using the "sudo" command.

STEP 7: Create New User Account

This new user account will serve as an alternative to the root user, enabling you to perform regular activities without the need for administrative permissions.

  • Enter a full name for the new user.
  • Enter a username for the new user account. The username should start with a lower-case letter, followed by any combination of numbers and more lower-case letters.
  • Enter a unique and strong password for the new user..

Step 8: Configure the Clock

Select the appropriate time zone. I selected Central for the Houston Office and click "continue".

STEP 9: Disk Partitioning

In this step, you will need to decide how you want to partition your disks in Debian 12. You have four options to choose from:

  • Guided Partition - Use entire disk: Selecting this option allows the system to automatically partition the entire disk for you.
  • Guided Partition - Use entire disk and set up LVM: This option automatically sets up Logical Volume Management (LVM) in addition to partitioning the entire disk. LVM provides flexibility in managing storage volumes.
  • Guided Partition - Use entire disk and set up encrypted LVM: If you require an extra layer of security, this option will automatically set up an encrypted LVM configuration along with partitioning the entire disk.
  • Manual partitioning: If you prefer to have complete control over the disk partitioning process, you can choose manual partitioning. This option allows you to customize and define the partitions according to your specific requirements.
  1. Here I am choosing Guided Partition - Use entire disk method and click "continue".
  2. Select the drive and click "continue".
  3. Select All files in one partition and click "continue".
  4. Review the Partions and select Finish partioning and write changes to disk and click "continue".
  5. After choosing the device, choose "Yes" to proceed with creating new partition table on the selected device. This action will initialize the disk with a fresh partition table, allowing you to define and allocate partitions as per your requirements. The Debian 12 base installation will begin now.

STEP 10: Configure Package Manager

After a few minutes, the installer will prompt you to scan for additional media. This scan is primarily performed to identify any supplementary media that can be utilized by the package manager (apt). In most cases, you will not have any additional media apart from the ISO image you initially booted from.

  1. I am choosing the "No" option to proceed to the next step, click "continue".
  2. Select a location of the server and click "continue".
  3. Select a mirror from the list of available Debian archive mirrors and click "continue".
  4. If your machine is configured to access the internet through a proxy, you will be prompted to enter the proxy information in this step. Provide the necessary proxy details, including the proxy server address and port number. If you do not have a proxy configured or do not require a proxy for internet access, you can leave the fields blank and proceed by pressing the "Continue" button.

STEP 11: Popularity Contest Configuration

  1. Click "No" and then "continue".

STEP 12: Software selection

  1. I am selecting to install Debian desktop environment, GNOME desktop environment, SSH server (this can also be added later see Enable SSH on Debian 12) and standard system utilities.
  2. Click "continue".

STEP 13: GRUB Installation

  1. Choose "Yes" and press the "Continue" button.
  2. Select the drive on which you want to install the boot loader and click "continue".
  3. After the installation process is complete, your system will need to be rebooted. Remove the USB Bootable Drive and click "continue". Upon reboot, you will be presented with a login prompt where you can enter the username and password you created during the installation.
  4. Login to Debian 12

STEP 14: Debian Initial Setup

At first login, you will be greeted with the Gnome initial setup welcome message.

  1. Click Next to "continue".
  2. Select your preferred keyboard layout or an input method and click Next.
  3. Next, you will be prompted to enable location services on your Debian 12 system. Toggle the switch to OFF position and click "continue".
  4. Connect your online accounts, such as Google, Microsoft, if you want to easily access your email, contacts, documents and photos. Click "Skip".
  5. GNOME initial setup is complete! Click "Start Using Debian GNU/Linux" to close this wizard.
  6. Run update
sudo apt update && upgrade -y

Set the Sleep Actions​

For GNOME, go to Settings > Power and adjust the sleep settings to off for the system. The sleep for the monitor can remain.

Enable SSH on Debian 12​

To enable SSH on Debian 12, you first need to install the OpenSSH server package. Here are the steps to follow:

Install OpenSSH Server: Run the following command to install the OpenSSH server:

sudo apt update && sudo apt install openssh-server

Check SSH Service Status: After installation, you can check if the SSH service is running with the command:

systemctl status ssh

Enable SSH Service: If the SSH service is not running, you can enable it to start on boot and start the service with:

sudo systemctl enable ssh
sudo systemctl start ssh

Verify SSH Configuration: You can edit the SSH configuration file located at /etc/ssh/sshd_config to make necessary changes, such as changing the default SSH port or disabling root login. Restart SSH Service: After making any changes to the configuration file, restart the SSH service to apply the changes:

sudo systemctl restart ssh

Connect to SSH Server: Use an SSH client to connect to your Debian 12 server. For example, from a terminal, you can use:

ssh username@your-server-ip

Troubleshoot SSH Connection Issues: If you encounter connection issues, ensure that your firewall is not blocking SSH connections and that the SSH service is running properly.

Add User as Super User​

To add a user as a superuser on Debian, follow these steps:

Create a New User: Use the adduser command to create a new user. For example:

sudo adduser username

Replace username with the desired username. This command will prompt you to set a password and provide additional information about the user. Add User to the Sudo Group: After creating the user, add them to the sudo group to grant superuser privileges. Use the usermod command:

sudo usermod -aG sudo username

Alternatively, you can directly create a new superuser with sudo privileges in one command:

sudo adduser username sudo

Verify the User's Groups: To ensure the user has been added to the sudo group, you can use the groups command:\

groups username

Test Superuser Privileges: Log in as the new user and test sudo privileges by running a command that requires elevated permissions:

sudo ls /root

If you are using a graphical interface, you can follow these steps:

  1. Create a New User: Use the adduser command as described above.
  2. Add User to the Sudo Group Using System Settings:
    • Go to the Applications menu and click on the Users option.
    • Unlock the user settings by entering the root password.
    • Click on the newly created user and convert it into an administrator account. These steps will ensure that the new user has superuser privileges and can perform administrative tasks on the Ubuntu system.
